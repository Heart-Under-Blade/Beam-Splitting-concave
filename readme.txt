Command line arguments:

1 - type of particle (int) (0 - Hexagonal prism, 1 - Concave hexagonal prism);
2 - half height (double)
3 - radius (double)
+4 - dept of cavity (double) (if type of particle is '1')
4 - the real part of refraction index (double) 
5 - gamma number of orientations (int) (must be uneven)
6 - beta number of orientations (int)
7 - size of the output array (int)
8 - number of the internal reflections (int)
9 - fixed of random rotation of particle due tracing
